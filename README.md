# enumWithSAS
Easy SAS tokens abuse tool

# Installation

```bash
git clone https://github.com/Invertebr4do/enumWithSAS.git
cd enumWithSAS && chmod +x enumWithSAS.py
pip3 install pwntools
```

# Usage

```bash
./enumWithSAS.py --help 
```
